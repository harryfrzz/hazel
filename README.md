# Hazel

A macOS live wallpaper app that plays videos as your desktop background.

## Features

- Set any video as your live wallpaper
- Loop or play once options
- Mute/unmute audio
- App Hides into menu bar (no dock icon)

## Installation

1. Download the latest release from GitHub
2. Open `hazel.dmg`
3. Drag and move the app to the Applications folder

**Note:** Hazel may not appear in Spotlight search. To open it, simply open Finder and navigate to the Applications folder, then double-click Hazel.

## Usage

- Click the menu bar icon to open the "manage" window
- Click "+" to add videos
- Select a video to set it as your wallpaper
- Right-click a wallpaper for options (loop, mute, remove)
- Use the menu bar to change wallpaper fit or toggle open on startup

## Requirements

- macOS 13.0 or later
